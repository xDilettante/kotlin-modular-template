# kotlin-modular-template
Мультимодульный шаблон Gradle Kotlin/JVM проекта с Version Catalog. Поддержка Ktor, Exposed ORM, gRPC, Telegram Bot API, BouncyCastle, Hoplite. Автоматическое управление JDK 21 (Foojay). Кастомное цветное логирование Logback. Модули: xLib (модуль библиотеки), xApp (модуль-пример приложения), xEmpty (шаблон для новых модулей).
